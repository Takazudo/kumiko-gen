# Installation

## npm (global)

```bash
npm install -g @takazudo/kumiko-gen
```

## npm (local)

```bash
npm install @takazudo/kumiko-gen
```

## pnpm

```bash
pnpm add @takazudo/kumiko-gen
```

## One-time use (npx)

```bash
npx @takazudo/kumiko-gen hello-world --out hello.svg
```
