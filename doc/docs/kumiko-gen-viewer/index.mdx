# kumiko-gen-viewer

kumiko-gen-viewer is an interactive web application for generating and customizing Japanese kumiko geometric patterns in real time.

**Live app**: [kumiko-gen.netlify.app/pj/kumiko-gen/](https://kumiko-gen.netlify.app/pj/kumiko-gen/)

## Features

- Real-time pattern preview with crossfade transitions
- 30+ color schemes (Dracula, Nord, Catppuccin, Gruvbox, TokyoNight, etc.)
- Per-layer customization (foreground color, stroke width)
- Adjustable divisions (6/8/10), zoom (1-10), and background color
- Random generation with one click
- Download optimized SVG
